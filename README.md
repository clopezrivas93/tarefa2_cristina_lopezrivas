📖 Descrición

Este proxecto emprega o paquete Faker para xerar datos ficticios de usuarios con fins de proba. Créanse 15 usuarios con información básica (nome, dirección, correo electrónico e teléfono) que se almacenan nun dicionario. Amósanse os datos de todos os usuarios e, posteriormente, selecciónase aleatoriamente un deles como afortunado.

⚙️ Requisitos

Python 3.x, Contorno virtual, Paquete Faker
